# Apple TV Sample

## Purpose
This PlayKit sample show how to create and setup PlayKit SDK player with Apple TV.

## How to use

1. Run `pod install`
2. Open generated `.xcworkspace` file with Xcode
3. Run on simulator or device (DRM content will only work on device).
